# DFW Live Toll Tracker & Predictor 🚗💰

A data pipeline and future web application designed to track and predict real-time toll prices in the Dallas-Fort Worth metroplex. 

This project aims to monitor both static NTTA routes (Dallas North Tollway, PGBT) and dynamic, traffic-based TEXpress lanes (LBJ, I-35W) to help drivers anticipate price spikes and make cost-effective routing decisions.

## 🚀 Project Roadmap

- [x] **Phase 1: Data Pipeline** - Python scripts to ping toll APIs/endpoints and fetch live rates.
- [ ] **Phase 2: Database Integration** - Store time-series toll data in a SQL database to track historical trends.
- [ ] **Phase 3: Web Dashboard** - A responsive user interface (Streamlit/React) to display current prices.
- [ ] **Phase 4: AI Prediction** - A PyTorch-based machine learning model to forecast short-term toll fluctuations based on time of day and traffic congestion.
